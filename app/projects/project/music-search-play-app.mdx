# Music/Search/Play App

This project utilizes MERN stack. It has the ability to search for music videos on YouTube and play them on the site. There is a search bar where you can type in the name of a song or artist and it will return a list of videos that match your search. You can click on any of the videos to play them on the site. I am quite proud of this one and will use it for my own music discovery.

Here is a link to the [GitHub repository](https://github.com/angel33la/wdv339-spotify-pp3) and a link to the live site [Music/Search/Play App](https://wdv339-spotify-pp3.onrender.com/) where you can find the code and explore how the music search and play functionality works in this project. Feel free to check it out and see how it works!
