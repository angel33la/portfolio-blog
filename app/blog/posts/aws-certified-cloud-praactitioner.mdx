## Course Setup at Full Sail University
---

Each week, the course is broken down into different topics and assignments that focus on cloud application development and AWS services.

- **Week 1**: Students learn about various cloud services, including IaaS, PaaS, SaaS, cloud services, and cloud providers. They also delve into the ethics of cloud computing.
- **Week 2**: This week builds on the previous week’s knowledge by exploring cloud computing software functions and serverless technology. Students learn about YAML, Lambda, SDK, and CDK.
- **Week 3**: Students utilize the latest industry APIs and deepen their understanding of cloud services. Themes include AppSync, Amplify, DynamoDB, and Cognito.
- **Week 4**: In the final week, students learn advanced JavaScript methods and apply them to their applications. Themes include SAM and facial recognition software.

The course has been straightforward so far, allowing me to use the AWS Academy as a valuable learning resource. I can progress through the modules at my own pace, which has been beneficial. The academy offers labs where I can practice what I’m learning, and I can take notes and review the material as needed, which helps me grasp the rules and concepts of cloud computing. Before this course, my knowledge of cloud computing was limited, but now I feel much more confident in my understanding of the basics and how it operates. I’m excited to continue learning and explore how I can apply this knowledge in the future!
